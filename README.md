# 🥗 SlimMoms – Smart Nutrition & Calorie Tracking Platform

SlimMoms is a modern full-stack health and nutrition platform that allows users to calculate their daily calorie needs, track their dietary routines, and receive personalized diet recommendations.

The project is built with a React + Redux frontend and a Node.js + Express + MongoDB Atlas backend architecture.

## 🚀 Key Features

### 1️⃣ Smart Product Selection
- AI-powered search when adding products
- Real-time suggestions as the user types
- Quick add with “Enter” key

### 2️⃣ Dynamic Daily Diary
- Calendar integration for tracking past and future dates
- Instant overview of daily calories and consumed products
- Responsive, mobile-first design

### 3️⃣ Personalized Calorie Calculator
- Calculates daily calories based on height, weight, age, and target weight
- Recommends foods based on blood type
- Automatic saving with Silent Save UX

### 4️⃣ Modern UI & Dark Mode
- Pixel-perfect Figma design implementation
- Glassmorphism visuals and animated elements
- Smooth button animations and transitions

### 5️⃣ Advanced Notification System
- Real-time feedback with react-hot-toast
- Instant alerts for successful, failed, or incomplete actions

## 🛠 Technology & Architecture

### Frontend
- **React (Vite):** fast and modern
- **Redux Toolkit:** centralized state management
- **Framer Motion:** smooth animations
- **i18next:** multi-language support (Turkish, English)
- **React Router:** secure routing
- **React Hot Toast:** modern notifications

### Backend
- **Node.js & Express:** scalable server architecture
- **MongoDB Atlas & Mongoose:** flexible NoSQL database management
- **JWT Authentication:** secure authentication and sessions
- **CORS & Morgan:** security and logging

### Additional Features
- IP whitelist and security settings
- Silent Save UX and data validation
- Clear error handling for API issues

## 📦 Installation & Running

1. **Install dependencies:**
   ```bash
   npm install
   cd backend && npm install
   ```
2. **Create a .env file:**
   ```env
   PORT=5001
   DB_HOST=<MongoDB Atlas Connection String>
   JWT_SECRET=<Strong Secret Key>
   ```
3. **Start the project:**
   ```bash
   npm run fullstack
   ```
   - **Frontend:** http://localhost:5173
   - **Backend API:** http://localhost:5001/api
